使用方法

基础用法

引入CSS和JS文件，组件会自动初始化

<!-- 引入样式文件 --> <link rel="stylesheet" href="css/language-selector-modern.css"> <!-- 引入脚本文件 --> <script src="js/language-selector-modern.js"></script>

自定义配置

通过配置选项自定义组件行为和外观

// 自定义初始化 window.SPADE_LANGUAGE_SELECTOR_MODERN.init({ mode: 'dropdown', // 'dropdown' | 'toggle' size: 'md', // 'sm' | 'md' | 'lg' draggable: true, // 是否可拖拽 autoClose: true, // 自动关闭下拉菜单 supportedLanguages: ['en', 'zh-Hans', 'es'] });
