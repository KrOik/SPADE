# SPADE 高级检索功能指南

## 概述

SPADE平台现已支持多维度智能检索功能，包括基础检索、活性类型检索、目标生物检索和结构特征检索，满足不同研究需求。

## 检索功能详解

### 1. 基础检索
- **ID检索**: 通过抗菌肽的唯一标识符进行精确检索
- **名称检索**: 通过抗菌肽的名称进行模糊匹配检索
- **序列检索**: 通过氨基酸序列片段进行检索
- **长度检索**: 通过序列长度范围进行检索（格式：10-50）

### 2. 活性类型检索
根据抗菌肽的抗菌谱进行分类检索：

- **革兰氏阳性菌 (Gram-positive)**: 针对革兰氏阳性细菌的抗菌肽
- **革兰氏阴性菌 (Gram-negative)**: 针对革兰氏阴性细菌的抗菌肽
- **真菌 (Fungal)**: 具有抗真菌活性的抗菌肽
- **病毒 (Viral)**: 具有抗病毒活性的抗菌肽
- **广谱抗菌 (Broad-spectrum)**: 对多种微生物都有活性的抗菌肽
- **抗真菌 (Antifungal)**: 专门针对真菌的抗菌肽
- **抗病毒 (Antiviral)**: 专门针对病毒的抗菌肽

### 3. 目标生物检索
根据抗菌肽的具体目标微生物进行检索：

- **金黄色葡萄球菌 (Staphylococcus aureus)**: 常见的革兰氏阳性病原菌
- **大肠杆菌 (Escherichia coli)**: 革兰氏阴性菌的代表
- **铜绿假单胞菌 (Pseudomonas aeruginosa)**: 重要的医院感染病原菌
- **肺炎克雷伯菌 (Klebsiella pneumoniae)**: 呼吸道感染病原菌
- **鲍曼不动杆菌 (Acinetobacter baumannii)**: 多重耐药菌
- **粪肠球菌 (Enterococcus faecalis)**: 革兰氏阳性球菌
- **白色念珠菌 (Candida albicans)**: 常见真菌病原体
- **烟曲霉 (Aspergillus fumigatus)**: 侵袭性真菌感染病原体
- **HIV病毒**: 人类免疫缺陷病毒
- **流感病毒**: 呼吸道病毒

### 4. 结构特征检索
根据抗菌肽的物理化学和结构特征进行检索：

#### 电荷特性 (Charge)
- **正电荷 (Positive)**: 带正电荷的抗菌肽
- **负电荷 (Negative)**: 带负电荷的抗菌肽
- **中性 (Neutral)**: 不带电荷的抗菌肽

#### 疏水性 (Hydrophobicity)
- **亲水性 (Hydrophilic)**: 亲水性强的抗菌肽
- **疏水性 (Hydrophobic)**: 疏水性强的抗菌肽
- **两亲性 (Amphipathic)**: 同时具有亲水和疏水区域的抗菌肽

#### 二级结构 (Secondary Structure)
- **α-螺旋 (Alpha-helix)**: 以α-螺旋为主要结构的抗菌肽
- **β-折叠 (Beta-sheet)**: 以β-折叠为主要结构的抗菌肽
- **无规卷曲 (Random coil)**: 无规卷曲结构的抗菌肽
- **混合结构 (Mixed)**: 具有多种二级结构的抗菌肽

## 使用方法

### 单一条件检索
1. 在相应的检索区域选择或输入检索条件
2. 点击"Search"按钮执行检索
3. 查看检索结果

### 多条件组合检索
1. 在多个检索区域同时设置条件
2. 系统将自动应用AND逻辑组合所有条件
3. 点击"Search"按钮执行检索
4. 查看符合所有条件的检索结果

### 清除检索条件
1. 点击"Clear Search"按钮
2. 所有检索条件将被清空
3. 显示所有抗菌肽数据

### 导出检索结果
1. 执行检索后，点击"Export CSV"按钮
2. 系统将导出当前检索结果为CSV文件
3. 文件包含所有检索到的抗菌肽信息

## 检索结果说明

检索结果表格包含以下信息列：

- **ID**: 抗菌肽的唯一标识符
- **Name**: 抗菌肽的名称
- **Sequence**: 氨基酸序列
- **Activity Type**: 活性类型
- **Target Organisms**: 目标生物
- **Structure**: 结构特征（电荷、疏水性、二级结构）
- **Action**: 操作按钮（查看详情、可视化）

## 技术特点

1. **实时检索**: 支持实时检索，响应速度快
2. **模糊匹配**: 支持部分匹配和模糊检索
3. **多维度**: 支持多个维度的组合检索
4. **用户友好**: 界面简洁，操作直观
5. **响应式设计**: 支持各种设备访问

## 注意事项

1. 检索条件为空时，系统将显示所有数据
2. 多个检索条件之间为AND关系
3. 序列检索支持部分序列匹配
4. 长度检索需要按照指定格式输入（如：10-50）
5. 检索结果支持分页浏览
