## Design

采用现代简约设计风格，使用纯原生HTML+CSS+JavaScript实现。支持下拉菜单和切换按钮两种交互模式，具备CSS3动画效果、响应式布局和无障碍访问功能，与SPADE系统现有设计语言保持一致
